# Part B: Full-Stack Documentation

This section details the Node.js backend and React/Vanilla frontend dashboard.

## Backend Features
- Express API with RESTful endpoints
- SQL Data Persistence
- JWT Authentication

## Frontend Features
- Interactive Dashboard (KPIs & Trends)
- D3.js Visualizations
- Data Filtering (Date, Channel, Language)

## Setup
```bash
cd backend
npm install
npm start
```
